Navigate to Class

Navigate to a class by name.

Command+O

1. Navigate to class
   Navigate to the MasterViewController class. Open the file by hitting enter.

2. Use CamelHumps to navigate
   Navigate to the DetailViewController class using CamelHumps (type "DVC")

3. Use middle matching to navigate
   Navigate to the any ViewController class by searching for "View"

4. Include non-project classes
   Check the "Include non-project classes" (press Command+O second time) checkbox and navigate to UIViewController
